# inProcess

Process and manage short note files.



## Installation

If you don't use `pipsi`, you're missing out.
Here are [installation instructions](https://github.com/mitsuhiko/pipsi#readme).

Simply run:

    $ pipsi install .


## Usage

To use it:

    $ inprc --help


## Tests

To run the tests use

    py.test --cov inprocess
